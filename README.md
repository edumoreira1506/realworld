# realworld-api

## Dependencies

```
NodeJS >= 10.17
Yarn >= 1.21
MongoDB >= 3.6
Nodemon >= 2.0
```

Clone the project.
```console
$ git clone git@github.com:edumoreira1506/realworld-api.git
$ cd realworld-api
```

Copy the contents of the `.env.example` to `.env` then change with the credentials of your local environment.

```console
$ cp .env.example .env
```

Install all packages and run.
```console
$ yarn
$ yarn start
```

Run lint.
```console
$ yarn lint
```
